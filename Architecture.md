## ğŸ¯ Executive Summary

Papillon AI is a VS Code extension that brings AI-powered development assistance directly into your coding environment. Think of it as having an expert developer sitting next to you, ready to explain code, suggest improvements, write tests, and automate tasks - all while you stay in your editor.

**What makes Papillon AI special:**
- Works with 40+ AI providers (Claude, GPT-4, Gemini, local models)
- Understands your entire codebase through semantic search
- Executes tasks safely with approval workflows
- Extends capabilities through MCP (Model Context Protocol)
- Enterprise-ready with governance and compliance features

### Layer Responsibilities

**Presentation Layer** (What you see)
- React-based UI components
- Chat interface
- Settings panels
- History views
- Real-time updates

**Orchestration Layer** (The coordinator)
- Provider manages everything
- Routes messages between UI and services
- Maintains conversation state
- Coordinates tool execution
- Handles approvals

**Service Layer** (The workers)
- API Handler: Talks to AI providers
- MCP Hub: Manages external tools
- Code Index: Semantic code search
- Terminal Service: Command execution
- Browser Service: Web testing
- Git Service: Version control
- Cloud Service: Sync and backup
- Telemetry: Usage analytics

**Integration Layer** (External connections)
- AI Providers: Claude, GPT-4, Gemini, etc.
- MCP Servers: Filesystem, databases, APIs
- Qdrant: Vector database for code search
- VS Code API: Editor integration

## Conclusion

Papillon AI is architected as a modular, extensible, and secure AI development assistant that integrates deeply with VS Code while maintaining flexibility to work with multiple AI providers and external tools.

**Key Architectural Principles:**

1. **Modularity**: Components are loosely coupled and independently replaceable
2. **Extensibility**: MCP protocol allows unlimited capability expansion
3. **Security**: Multiple layers protect against dangerous operations
4. **Performance**: Optimized for responsiveness and low resource usage
5. **Flexibility**: Works with 40+ AI providers and adapts to any workflow
6. **Enterprise-Ready**: Governance, compliance, and audit capabilities built-in

**The Big Picture:**

Papillon AI sits between you and your development tools, understanding your intent, coordinating with AI providers, and safely executing actions - all while staying out of your way and letting you focus on building great software.
